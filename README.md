# Weather App Project

This is a simple and interactive Weather Application built using HTML, CSS, and JavaScript. It uses a weather API to fetch real-time weather data and display it in a clean and modern user interface.


## 📝 What Does It Do?

The Weather App allows users to search for any city and instantly see the current weather conditions.
- Users can search for a city using the search bar
- Displays the current temperature of the selected city
- Shows weather conditions with an icon (rain, clouds, etc.)
- Displays additional weather information such as humidity and wind speed
- Updates the weather data dynamically using an API
  

## ⚙️ How Does It Work?

The project is built using core JavaScript concepts along with API integration:
- **HTML Structure:** Creates the layout for the search bar, weather icon, temperature display, and weather details
- **CSS Styling:** Provides a modern gradient background, responsive layout, and clean UI components
- **API Integration:** Fetches real-time weather data from a weather API
- **DOM Manipulation:** JavaScript dynamically updates the temperature, weather icon, humidity, and wind speed
- **Event Handling:** The search button triggers the API request when the user searches for a city
